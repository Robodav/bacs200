Skill Bars Tutorial

UNC - BACS 200

This is a rather simple addition that you can apply to your resumes and other projects. Skill bars are intended to show mastery over certain skills using progress bars with percent values.

HTML

90%

CSS

80%

JavaScript

65%

PHP

60%

The first thing you'll need to do is enter the HTML: 

			<p>HTML</p>
			<div class="container">
			<div class="skills html">90%</div>
			</div>

			<p>CSS</p>
			<div class="container">
			<div class="skills css">80%</div>
			</div>

			<p>JavaScript</p>
			<div class="container">
			<div class="skills js">65%</div>
			</div>

			<p>PHP</p>
			<div class="container">
			<div class="skills php">60%</div>
			</div>

Separate paragraphs with their own divs allow for unique styling and colors for each bar.

Now, the CSS will control those styles: 

			/* Make sure that padding behaves as expected */
			* {
				box-sizing:border-box
			}
			
			/* Container for skill bars */
			.container {
				width: 100%; /* Full width */
				background-color: #ddd; /* Grey background */
			}
			
			.skills {
				text-align: right; /* Right-align text */
				padding: 10px; /* Add some padding */
				color: white; /* White text color */
			}
			
			.html {width: 90%; background-color: #4CAF50;} /* Green */
			.css {width: 80%; background-color: #2196F3;} /* Blue */
			.js {width: 65%; background-color: #f44336;} /* Red */
			.php {width: 60%; background-color: #808080;} /* Dark Grey */

As you can see, this uses CSS tricks to display boxes overlayed with their own backgrounds. The style here is especially important, differing color and making sure the skill names are prominent.

Press Ctrl + U and take a look at the CSS file skill.css, specifically the bottom section, where the parts that style the bars are located. You can see that in just a few lines, a great looking set of skill bars can be created! Check out the tutorial page at W3Schools for more information.
